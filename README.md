# Iris-Flower-Classification-using-Machine-Learning
A machine learning project that uses the Random Forest Classifier in Python (scikit-learn) to classify Iris flower species based on sepal and petal measurements, including model training, prediction, and evaluation.

**Overview**

This project demonstrates a machine learning classification model built using the Iris dataset. The model uses the Random Forest Classifier from scikit-learn to classify iris flowers into one of three species based on their sepal and petal measurements.

**Features**
Data loading and exploration
Data preprocessing
Model training using Random Forest
Prediction on new samples
Model evaluation using accuracy score
Simple and beginner-friendly implementation
Technologies Used
Python
Pandas
NumPy
Scikit-learn
Jupyter Notebook
Dataset

The project uses the built-in Iris dataset available in the scikit-learn library. It contains 150 samples of three iris species:

Setosa
Versicolor
Virginica

Each sample includes:

Sepal Length
Sepal Width
Petal Length
Petal Width
Machine Learning Algorithm

The model is trained using the Random Forest Classifier, an ensemble learning algorithm that combines multiple decision trees to improve classification accuracy and reduce overfitting.

**Results**

The trained model successfully classifies iris flower species with high accuracy on the test dataset.
